## Run Locally

Clone the project

```bash
  git clone https://github.com/SuperninjaXII/linuxmint.github.io
```

Go to the project directory

```bash
  cd linuxmint.github.io 
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
 pnpm run docs:dev
```
